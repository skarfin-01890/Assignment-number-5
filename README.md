1️⃣ What is the difference between var, let, and const?
Answer:The difference between var, let, and const are var and let can be re-assigned and var can also be used for re-declared but let can't. And, the variable which declare by const can't be re-declared and re-assigned.

2️⃣ What is the spread operator (...)?
Answer:The spread operator  is used to expand or spread elements of an array or object.
It takes values from an array or object and spreads them individually.

3️⃣ What is the difference between map(), filter(), and forEach()?
Answer:The difference between map(), filter(), and forEach() are map is used for modifying and transforming an element.It doesn't change the length of main array. The filter is used for selecting an element by given condition.It returns a new array and can change the length of an array. And, the for each is used to loop through an array.

4️⃣ What is an arrow function?
Answer:Arrow function is the short and modern way to write a function of Javascript. It doesn't take function keyword and takes small syntax.That's why it is very useful than normal function.

5️⃣ What are template literals?
Answer: Template literals is the modern way of write a string. Backtick is used to write template literals. By using template literals we can write multiline string , variable and expression.
